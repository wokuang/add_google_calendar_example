# add_google_calendar_example
Use python to add google calendar

## 前置動作
有先建立 python [virtualenv](https://virtualenv.pypa.io/en/latest/)，先進入 gcal 在開始使用
``` bash
  $ lsvirtualenv
  $ workon gcal
```
